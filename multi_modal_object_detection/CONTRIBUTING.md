**Contributing Guidelines**

Thank you for your interest in contributing to this project. Please follow these steps:

- Fork the repository and create a feature branch (`feature/your-feature`).
- Make small, focused commits with clear messages.
- Ensure your code follows PEP8 and project style.
- Run tests locally: `pytest`.
- Open a pull request describing your changes and linking any relevant issues.

For major changes, open an issue first to discuss the design.
